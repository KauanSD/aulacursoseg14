Adobe DreamWeaver

O que é e para que serve o DreamWeaver?

É um programa de design web que permite criar, editar e gerenciar sites. Ele combina uma interface visual com um editor de código.

É possível criar, editar, gerenciar e modificar sites e aplicações web. Ele é um programa de design web que suporta HTML, CSS, JavaScript e outros padrões da web.

Como trabalhamos com o DreamWeaver?

Podemos cria os conteúdo do site incorporando os elementos por itens nos menus (COMANDOS) , de forma visual, ou criarmos o site programado (TAGs) com códigos em HTML, CSS e outras linguagem de programação web.
